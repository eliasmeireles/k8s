API Documentation

POST /ingress

Create a new ingress configuration.

Request Body:

  • name (string): Unique name for the ingress (required)
  • host (string): Hostname or IP to route traffic (required)
  • protocol (string): Protocol (http/https/tcp/udp, default: http)
  • upstream (string): Upstream server address (required)

Curl Example:

curl -X POST http://localhost:8080/ingress \ -H "Content-Type: application/json" \ -d '{"name": "example", "host": "example.com", "protocol": "http", "upstream": "http://127.0.0.1:8080"}'

Response:

JSON object indicating success or error.
